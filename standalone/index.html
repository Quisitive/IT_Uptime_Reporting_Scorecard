ⓘ Collection Sources & Scoring

Where each metric comes from, and how the health index is calculated.
Where each number comes from
Endpoint Security — Defender monthly activity report → “Total OverWatch-analyzed events” (Data Points). Escalations = cases opened to inform of an alert (search “Defender” in the ticket queue, last month). Interventions = tickets requiring action as an incident.

Intrusion Detection System — Perch Hero Dashboard, range “Last Month”. SOC activity → “Analyzed Logs” (Data Points), “Escalated Alerts” (Escalations). Interventions = escalations that became incidents.

Microsoft Defender — Incidents dashboard, 1st of last month 00:00 → 1st of this month 00:00. Data Points = incidents (status New/In Progress/Resolved; severity Low/Medium/High). Escalations = incidents acted on (severity Medium/High). Interventions = alerts marked True Positive requiring action.

Disk — Production & Backup SANs, via array interface.
How the Overall Health Index is scored (adjustable)
The index is a transparent weighted blend. Adjust weights below; they are saved with your settings.
• Uptime sub-score = System Uptime % (degraded hours count as half-down).
• Disk sub-score = 100 − highest SAN used% (worst array drives the score).
• Security sub-score = 100 − (5 × total interventions) − (1 × total escalations), floored at 0.
Weights are normalized automatically if they don't total 100.
